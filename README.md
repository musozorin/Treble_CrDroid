# Treble_CrDroid

Unofficial **crDroid 10.13** Generic System Image (GSI) for Treble-enabled devices.

# UNPACK USING 7z-zstd

## ✨ Project Highlights
* **Base:** Android 14 (crDroid 10.13)
* **Status:** Unofficial
* **Focus:** Minimalist, no telemetry, and bloat-free performance.
